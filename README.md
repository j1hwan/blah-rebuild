# 익명 질문&답변 서비스
[blahx2
](https://blahx2.totu.dev/) 강의를 참고하여 동일하게 구현

## 주요 기능
소셜로그인 - 구글  
익명 질문 등록  
익명 댓글 답변

## 기술 환경
**데이터베이스, 인증** - Firebase: Firestore, Authentication  
**UI** - Chakra UI

## .env
```

FIREBASE_AUTH_HOST={프로젝트id}.firebaseapp.com
```
